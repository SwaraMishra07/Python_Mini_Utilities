# Code of Conduct

## Our Pledge
We, as contributors and maintainers of **Python Mini Utilities Hub**, pledge to create a welcoming, respectful, and harassment-free environment for everyone participating in this project.

This repository is designed to help beginners learn Python and open-source collaboration. We are committed to making this a safe, friendly space where learning and growth are encouraged.

---

## Our Standards

### Expected behavior
Participants are expected to:
- Be respectful, patient, and supportive of others
- Use inclusive and beginner-friendly language
- Provide constructive feedback focused on learning and improvement
- Respect different experience levels and learning speeds
- Help others when possible and encourage questions

### Unacceptable behavior
Unacceptable behavior includes, but is not limited to:
- Harassment, discrimination, or personal attacks
- Mocking beginners or discouraging questions
- Insulting, rude, or aggressive comments
- Publishing private information without permission
- Any behavior that makes others feel unsafe or unwelcome

---

## Maintainer Responsibilities
Project maintainers are responsible for:
- Enforcing this Code of Conduct fairly and consistently
- Reviewing issues and pull requests respectfully
- Encouraging beginner contributions
- Taking appropriate action when violations occur

Maintainers have the right to remove, edit, or reject contributions that violate this Code of Conduct.

---

## Scope
This Code of Conduct applies to:
- GitHub repositories, issues, pull requests, and discussions
- All interactions related to this project, whether online or offline

---

## Enforcement
If you experience or witness behavior that violates this Code of Conduct, please report it to the project maintainers.

All reports will be reviewed promptly and handled with confidentiality.  
Actions may include warnings, temporary restrictions, or permanent removal from the project depending on severity.

---

## Community Values
This project values:
- Learning over perfection
- Kindness over ego
- Progress over pressure
- Collaboration over competition

By participating in this project, you agree to uphold these values.
